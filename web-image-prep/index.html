Drop · Pick · Send

  1. Pick a profile, tune, hit Download .bmp.
  2. Copy the file to SD/apps_data/tagtinker/dropped/ on your Flipper.
  3. Open TagTinker → Targeted Payloads → your tag → Set Image.
  4. The Flipper lists every BMP whose resolution matches that tag — pick a page, then send.

Filenames are resolution-shaped only, so one BMP can be sent to any matching tag, on any page.
